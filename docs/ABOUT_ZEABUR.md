# 关于 Zeabur：你没看错

2026 年左右 Zeabur 的模式已经变成：

1. **购买他们合作的服务器**，或  
2. **绑定你自己已有的服务器**（阿里云 / 腾讯云等）

Free 方案主要是「免费用控制台管 **1 台自有机器**」，**不是**再送一台免费云主机跑你的 API。

所以你在页面上看到「购买新服务器 / 连接自己的服务器」——**是正常的**。

---

## 你该怎么选

| 情况 | 建议 |
|------|------|
| 已有阿里云 2 核 2G | **直接部署后端**，见 [`DEPLOY_ALIYUN.md`](./DEPLOY_ALIYUN.md)。不必再买 Zeabur 机器。 |
| 想用 Zeabur 面板 | 把阿里云「添加为自有服务器」，再在上面部署；机器还是你的 2C2G。 |
| 没有云服务器 | 别买 Zeabur 新机（要钱）；用 Neon + 本机 cpolar 演示，或买国内学生机。 |

---

## 2 核 2G 能不能跑？

**能。** YangMind 这种 FastAPI：

- 数据库在 **Neon**（不占阿里云内存）  
- 服务器只跑 **uvicorn**（通常远小于 2G）

真正吃内存的是「一台机器同时跑 Postgres + 很多服务」。你把库放 Neon，2G 很宽裕。
